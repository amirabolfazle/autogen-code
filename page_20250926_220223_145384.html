Neon UI
A responsive neon-themed animated demo
Created: 2025-09-26 22:01:26

Glow Header

Animated neon heading with layered shadows and gradient glow. Smooth slide-up effect on scroll for that cinematic entrance.

Responsive Grid

Three-to-one column layout adapts across devices. Cards animate into place with a natural easing curve and subtle hover lift.

Accessible Motion

Respects prefers-reduced-motion. Animations gracefully disable when reduced-motion is requested to avoid discomfort.

Animated Orbs

Soft blurred orbs add depth and color without distracting from the content—the neon palette keeps a futuristic tone.

Slide-up Reveal

Each card uses IntersectionObserver to trigger a slide-up + fade-in animation when entering the viewport.

Footer Link

Footer contains a GitHub link with icon and attribution — perfect for sharing the source or portfolio link.
